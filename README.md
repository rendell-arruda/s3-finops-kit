# s3-finops-kit

> FinOps toolkit focused on AWS S3 — collect storage data, visualize costs on an interactive dashboard, and get AI-powered recommendations.

## What it does today

- Collects storage data from all S3 buckets across multiple regions
- Identifies the storage class of each bucket via CloudWatch (Standard, IA, Glacier, etc.)
- Exports data to CSV
- Displays everything on a public interactive dashboard with charts, cards, and filters

## Project structure

```
s3-finops-kit/
├── report.py        # collects S3 data and exports to data.csv
├── data.csv         # generated by report.py (updated by Actions)
├── index.html       # public dashboard — GitHub Pages
├── requirements.txt
├── .env.example
└── README.md
```

## Setup

```bash
git clone https://github.com/rendell-arruda/s3-finops-kit.git
cd s3-finops-kit
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # fill in your AWS_PROFILE
python report.py
```

## Requirements

- Python 3.12+
- AWS credentials configured (`~/.aws/credentials`)
- Permissions: `s3:ListAllMyBuckets`, `s3:GetBucketLocation`, `cloudwatch:GetMetricStatistics`
